# Spec Summary (Lite)

Make BookmarkFeeder self-hostable on a NAS via Docker Compose with a clean
multi-container topology: a YARP reverse-proxy gateway is the single external entry
point that routes `/api` to the API and everything else to a static web container,
with PostgreSQL behind them. The same gateway runs under Aspire in dev, so routing is
identical across environments - no CORS for the web app, no frontend path rewrites.
The API is hardened for production (forwarded headers, opt-in HTTPS redirect, exposed
health probes, per-endpoint rate limiting), compose artifacts are generated by Aspire
publish, and the data layer is validated with Testcontainers. TLS termination is
deferred (the gateway serves plain HTTP).
